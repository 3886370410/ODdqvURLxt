# 项目简介

## 核心功能
- 基于YOLOv8的汽车表面损伤检测系统
- 支持图片、视频及摄像头进行实时检测
- 提供17类汽车表面损伤识别：车身面板凹陷、挡风玻璃损坏、灯具损坏、保险杠凹陷等
- 支持批量检测并查看所有结果
- 支持检测结果保存功能
- 可自定义置信度和IOU阈值
- 支持选择显示标签和原图
- 可选择不同的检测模型

## 适用场景
- 汽车维修厂进行车身损伤快速检测
- 二手车评估中对车辆状况的初步判断
- 汽车制造过程中的质量检测
- 保险公司对车辆定损的辅助评估
- 车主对自己车辆状况的自检

## 技术特点
- 使用Pycharm+Anaconda或VSCode+Anaconda开发环境
- 基于Python 3.9，集成opencv-python、PyQt5、ultralytics、torch1.9等技术栈
- 提供完整的UI界面和交互功能
- 包含3072张标注数据集用于模型训练
- 预训练模型可直接使用，配置成功环境即可运行
